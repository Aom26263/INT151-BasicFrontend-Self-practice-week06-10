ข้อ 1 มีการใช้ higher order function คือมี 
- function ซ้อน function โดย function เเรกจะเป็นการสร้าง account เป็น object ว่างๆเเละกำหนดเลขบัญชีเป็น 1 
- function ถัดมาจะเป็นการสร้างบัญชีโดยข้างในบัญชีจะมีเลขบัญชีรันตามลำดับเเละจำนวนเงินตามที่กำหนด 
- function ถัดมาจะเป็นการฝากเงินโดยกำหนด acc เก็บข้อมูล บัญชี ID เเละมีเช็คว่าถ้าไม่ใข้ acc return account not found ถ้ามีก็เอาเงินที่มีอยู่ไปบวกกับจำนวนที่ฝากมาเเละ
return ออกมาเป็นค่าเงินที่ถูกฝากเเล้ว
- function ถัดมาจะทำงานเหมือนกับ function ที่ 3 เเต่อันนี้เป็นการถอนเงิน ก็จะนำยอดเงินในบัญชีเราไปหักลบกับเงินที่เราถอนไป return ค่าเงินคงเหลือของเรา
- function ถัดมาจะไว้เช็คเงินคงเหลือในบัญชี
- function ถัดมาเช็คภาพรวมบัญชีทั้งหมดของเรา โดยใช้ .map ในการ map ค่า id กับ balance 
เเละสุดท้ายก็ return function ทั้งหมด เพื่อให้ function มันทำงาน
